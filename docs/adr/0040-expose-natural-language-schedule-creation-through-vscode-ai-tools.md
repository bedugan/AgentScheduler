# Expose natural-language schedule creation through VS Code AI tools

AgentScheduler will expose schedule creation through a VS Code language model tool so Copilot agent mode can create schedules from natural-language requests, with an explicit chat participant and slash command as fallback surfaces. We chose this because users should be able to say "create a scheduled task that runs every hour to..." without walking through a wizard, while still retaining the Codex-like schedule detail for review and editing.
